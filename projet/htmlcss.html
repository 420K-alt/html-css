Introduction au html/css

Introduction Html et CSS

Pour cette premiere introduction nous allons vous presentez moi et mon camarade tres brievement a quoi sert le html et le Css. Le Html est un language qui sert principalement a construire le squelette d'un site web. Le Css est la pour accompagner le Html.
Sans le Html, le Css ne sert a rien ou presque.


Comprendre le HTML :

Les bases du html. Pour comprendre plus facilement le html, on peut comparer un script (de html) comme une sorte de grande boite ou nous mettons plein de petite boite, on appelle sa des balises. Dans l'exemple ci-dessous, nous avons donc le squelette de base, qui permet au script de creer une page web localement et dans les balise body des elements rajoutte pour mettre du contenu.




  <!DOCTYPE html>
  <html lang="en">
  <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Document</title>
  </head>
  <body>
      Mon premier texte
  </body>
  </html>
