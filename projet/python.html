Introduction au Python

Introduction python

Pour cette premiere lecon de python, si vous etes un debutant ou quelqun qui manipule le python de maniere ocasionelle nous allons essayer de vous expliquer comment sa marche, les bases puis les commandes a apprendre par coeur et qui nous server dans presque tout les programmes.



Print() et operations :

Pour cette premiere lecon, nous allons etudier enssemble la base des print(), des calcul, des signes a connaitre et aussi quelque petit conseil pour ne pas vous degouter. Premierement les print(), comment sa marche? Alors c'est tout simple, print() est un outil permettant d'ecrire dans la console python ce que vous voulez qu'il en ressorte ou encore des resultats que vous avec calculer auparavant dans votre code. Petit rappel, Output signifie la sortie soit ce que le code va donner.



print("Hello world")

>> Output : Hello world


Dans notre exemple ci dessus, il y'aura ecrit Hello world.


Maintenant que nous avons vu la base, je vais expliquer les principale operations et leur signe particulier:
L'adition: rien de plus simple, nous utilisont le + pour adissioner de valeur.
La soustraction: c'est aussi la meme chose que en maths soit -
La multiplication: le signe est diferent, ce n'est plus un x mais une etoile (*)



Les variables :

Une variable est un concept tres important car on le retrouve dans presque tout les languages et scripts, peu importe le theme. Une variable est une donnee d'un programme qui permet d'etre utilise a plusieurs reprises et/ou a faire des calculs. C'est interressant de pouvoir calculer une valeur mais si on ne la stocke pas, alors cela est inutile.

a=0
for i in range(3):
   b=a+1
   print("Je sais compter jusqu'a", b)

>> Output : Je sais compter jusqu'a 1
>> Output : Je sais compter jusqu'a 2
>> Output : Je sais compter jusqu'a 3
